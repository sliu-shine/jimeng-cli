# Selenium 抖音采集功能 - 使用说明

## ✅ 已完成的功能

### 1. 新增"Selenium 采集" Tab
在 Web 界面中添加了基于 Selenium 自动化的抖音视频采集功能，包括：

- **自动化浏览器**：使用 Selenium 控制 Chrome 浏览器自动采集
- **多账号批量采集**：支持同时采集多个抖音账号的视频
- **实时进度显示**：显示每个账号的下载状态和进度
- **智能筛选**：按点赞数筛选爆款视频
- **自动分类**：可选按视频标签自动分类整理
- **任务控制**：支持启动、停止下载任务

### 2. 与原有功能的区别

| 功能 | 原"抖音采集" Tab | 新"Selenium 采集" Tab |
|------|-----------------|---------------------|
| 实现方式 | API 调用 | Selenium 浏览器自动化 |
| 需要登录 | 否 | 需要首次手动登录 |
| 稳定性 | 依赖 API 可用性 | 更稳定，模拟真实用户 |
| 速度 | 较快 | 较慢（需要加载页面）|
| 功能 | 下载+转录+分析 | 仅下载视频 |
| 适用场景 | API 可用时 | API 不可用或需要更稳定时 |

## 🚀 使用步骤

### 首次使用（需要登录抖音）

1. **启动界面**
   ```bash
   bash start_viral_agent.sh
   ```

2. **打开浏览器**
   访问 http://127.0.0.1:7860

3. **进入 Selenium 采集 Tab**
   - 点击"🤖 Selenium 采集"

4. **首次登录**
   - 点击"🚀 开始下载"
   - 浏览器会自动打开抖音页面
   - **手动登录抖音账号**（扫码或密码登录）
   - 登录状态会被保存，下次无需重复登录

### 日常使用

1. **输入账号链接**
   ```
   https://www.douyin.com/user/MS4wLjABAAAA...
   https://www.douyin.com/user/MS4wLjABAAAA...
   ```
   每行一个链接

2. **配置参数**
   - **保存路径**：默认 `./douyin_videos`
   - **最低点赞数**：只下载点赞数超过此值的视频（默认 100000）
   - **滚动次数**：页面滚动次数，越多采集越多（默认 5）
   - **按标签分类**：自动按视频标签创建子目录

3. **开始下载**
   - 点击"🚀 开始下载"
   - 查看实时日志和进度
   - 浏览器会自动打开并开始采集

4. **停止下载**
   - 如需中途停止，点击"⏹️ 停止下载"

## 💡 使用技巧

### 1. 首次登录优化

- **保持登录**：首次登录后，浏览器会保存登录状态
- **用户数据目录**：`~/.douyin_selenium_profile/DouyinDownloader`
- **清除登录**：删除上述目录可重新登录

### 2. 采集效率优化

- **滚动次数**：根据账号视频数量调整（5-10 次通常足够）
- **点赞数筛选**：设置合理的阈值，避免下载低质量视频
- **分批采集**：一次采集 3-5 个账号，避免浏览器卡顿

### 3. 浏览器设置

- **无头模式**：默认显示浏览器窗口，可在代码中启用无头模式
- **反检测**：已配置反自动化检测，模拟真实用户行为
- **性能优化**：自动管理 ChromeDriver 版本

### 4. 错误处理

- **登录失效**：重新运行任务，手动登录
- **网络超时**：检查网络连接，重试
- **浏览器崩溃**：重启任务，浏览器会自动重新初始化

## 🐛 常见问题

### 1. 浏览器无法启动？

**原因**：ChromeDriver 版本不匹配或未安装

**解决**：
```bash
# 安装 webdriver-manager
pip install webdriver-manager

# 会自动下载匹配的 ChromeDriver
```

### 2. 需要重新登录？

**原因**：登录状态过期或用户数据目录被清除

**解决**：
- 重新运行任务
- 在打开的浏览器中手动登录
- 登录状态会自动保存

### 3. 下载速度慢？

**原因**：
- 网络速度限制
- 页面加载时间长
- 滚动次数过多

**优化**：
- 减少滚动次数
- 提高点赞数阈值
- 分批采集

### 4. 视频下载失败？

**原因**：
- 视频链接失效
- 网络中断
- 抖音反爬限制

**解决**：
- 检查网络连接
- 降低采集频率
- 使用代理（如需要）

### 5. 如何停止正在运行的任务？

点击"⏹️ 停止下载"按钮，任务会在当前视频下载完成后停止。

### 6. 下载的视频在哪里？

默认保存在 `./douyin_videos` 目录，如果启用了"按标签分类"，会按视频标签创建子目录。

## 🔧 技术实现

### 核心组件

1. **DouyinSeleniumDownloader**：Selenium 下载器类
   - 位置：`douyin/douyin_selenium.py`
   - 功能：浏览器控制、视频下载、进度回调

2. **任务管理**：
   - 全局变量 `selenium_task` 管理任务状态
   - 后台线程执行下载任务
   - 定时器刷新状态显示

3. **回调机制**：
   - `progress_callback`：更新下载进度
   - `log_callback`：记录日志信息

### 文件修改

- `viral_agent_ui.py`：
  - 添加 Selenium 采集 Tab
  - 实现任务控制函数
  - 集成实时状态显示
