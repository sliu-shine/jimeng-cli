# 快速开始

## 1️⃣ 安装依赖（首次使用）

```bash
# 安装所有依赖
pip install -r requirements_viral.txt
pip install -r requirements_douyin.txt

# 配置 API Key
export ANTHROPIC_API_KEY="your-claude-api-key"

# 验证安装
python3 setup_check.py
```

## 2️⃣ 采集抖音爆款视频

```bash
# 运行 Selenium 自动化脚本
python3 test_douyin_selenium.py
```

**操作步骤：**
1. 浏览器自动打开抖音用户主页
2. 等待页面加载完成
3. 点击右侧面板的「🔍 扫描爆款视频」
4. 确认视频列表
5. 回到终端按 Enter
6. 自动下载到 `douyin_videos/` 目录

## 3️⃣ 导入知识库

```bash
# 使用 Whisper 本地转录（免费但较慢）
python3 import_videos.py ./douyin_videos

# 或使用 Groq API（快 10-20 倍，推荐）
python3 import_videos.py ./douyin_videos --method groq --groq-api-key "your-key"
```

这个脚本会：
- ✅ 自动转录视频为文字
- ✅ 使用 Claude 分析爆款模式
- ✅ 存入向量知识库

## 4️⃣ 生成爆款文案

```bash
# 启动 Web 界面
python3 viral_agent_ui.py

# 访问 http://127.0.0.1:7860
```

或使用命令行：
```bash
python3 -m viral_agent generate "如何提升工作效率" --num 3
```

## 📊 查看知识库统计

```bash
python3 -m viral_agent stats
```

## 🔧 故障排查

**问题：扫描不到视频**
- 打开浏览器控制台（F12）查看错误
- 更新 Tampermonkey 脚本到最新版本

**问题：转录失败**
- 检查 ffmpeg 是否安装：`ffmpeg -version`
- 尝试使用 Groq API 代替 Whisper

**问题：生成质量不好**
- 增加知识库样本数量（建议 > 20 个）
- 确保样本是真实的爆款视频（高点赞数）
