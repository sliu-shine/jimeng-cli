# 转录本地视频功能 - 使用说明

## ✅ 已完成的功能

### 1. 新增"转录本地视频" Tab
在 Web 界面的第一个 Tab 中添加了本地视频转录功能，包括：

- **媒体扫描**：自动扫描指定目录下的 `.mp4`、`.m4a`、`.mp3`、`.aac`、`.wav`
- **目录筛选**：按 `douyin_videos/账号名/视频标题/文件` 的一级账号目录筛选
- **状态显示**：显示每个视频的转录状态（✅ 已转录 / ⏳ 未转录）
- **批量选择**：通过复选框选择需要转录的视频
- **智能跳过**：自动跳过已转录的视频
- **实时日志**：显示转录进度和结果预览
- **自动保存**：逐字稿保存为 `.transcript.json` 文件到媒体同目录
- **兼容导入**：命令行导入同时识别 `.transcription.json`、`.transcript.json`、`.txt`

### 2. 支持三种转录方式

- **云雾 API**（推荐）：速度快，国内访问友好
- **Groq API**：速度快，需要 API Key
- **Whisper 本地**：免费但需要 GPU

### 3. 界面优化

- 默认视频目录：`./douyin_videos`
- 默认 API Key：已预填云雾 API Key
- 视频列表显示：文件名、大小、状态、路径
- 目录筛选：自动读取 `douyin_videos` 下的账号/频道文件夹
- 转录日志：实时显示进度和统计信息

## 🚀 使用步骤

### 方式一：Web 界面（推荐）

1. **启动界面**
   ```bash
   bash start_viral_agent.sh
   ```

2. **打开浏览器**
   访问 http://127.0.0.1:7860

3. **转录视频**
   - 进入"🎬 转录本地视频" Tab
   - 确认视频目录路径（默认 `./douyin_videos`）
   - 可在"目录筛选"中选择单个账号/频道目录
   - 点击"🔍 扫描视频"
   - 勾选需要转录的视频
   - 选择转录方式（推荐"yunwu"）
   - 点击"🚀 开始转录选中视频"

4. **查看结果**
   - 转录完成后，逐字稿保存为 `视频名.transcript.json`
   - 视频列表会自动刷新，显示最新状态

### 方式二：命令行

```bash
# 批量转录整个目录
python douyin/douyin_cli.py transcribe \
  --video-dir ./douyin_videos \
  --method yunwu \
  --yunwu-api-key "your-yunwu-api-key"

# 只预检已有转录能否导入知识库，不调用 AI、不写入知识库
python douyin/import_videos.py douyin_videos --no-transcribe --dry-run

# 导入已有 Whisper API/云端转录结果到知识库，不重新转录
python douyin/import_videos.py douyin_videos --no-transcribe
```

## 📄 逐字稿格式

支持的逐字稿文件名：

- `视频名.transcription.json`：Whisper API 等外部转录结果
- `视频名.transcript.json`：Web UI/本项目转录结果
- `视频名.txt`：纯文本逐字稿

`.transcript.json` / `.transcription.json` 文件格式：

```json
{
  "text": "完整的逐字稿文本内容...",
  "segments": [
    {
      "start": 0.0,
      "end": 2.5,
      "text": "第一段文本"
    },
    {
      "start": 2.5,
      "end": 5.0,
      "text": "第二段文本"
    }
  ]
}
```

## 💡 使用技巧

### 1. 批量转录优化

- **分批处理**：一次选择 10-20 个视频，避免一次性处理太多
- **网络稳定**：使用云雾 API 时确保网络稳定
- **错误重试**：转录失败的视频可以单独重新选择转录

### 2. 转录质量优化

- **音频质量**：确保视频音频清晰
- **中文优化**：云雾 API 对中文识别效果最好
- **时长限制**：单个视频建议 < 30 分钟

### 3. 知识库管理

- **定期清理**：删除低质量或重复的样本
- **分类管理**：使用"赛道"标签分类（情感、干货、美食等）
- **质量优先**：优先导入高点赞、高质量的爆款文案

## 🐛 常见问题

### 1. 扫描不到媒体？

- 检查目录路径是否正确
- 确认格式为 `.mp4`、`.m4a`、`.mp3`、`.aac` 或 `.wav`
- 检查文件权限
- 如果刚采集完新账号，点击"刷新目录"更新目录筛选下拉框

### 2. 转录失败？

- 检查 API Key 是否正确
- 检查网络连接
- 查看错误日志，确认具体原因
- 尝试更换转录方式

### 3. 已转录的视频如何重新转录？

- 删除对应的 `.transcript.json` 文件
- 重新扫描视频列表
- 选择该视频进行转录

### 4. 如何批量导入到知识库？

可使用 Web UI 勾选导入，也可用命令行批量导入：

```bash
# 推荐先预检，确认可导入数量
python douyin/import_videos.py douyin_videos --no-transcribe --dry-run

# 正式导入，不重新转录
python douyin/import_videos.py douyin_videos --no-transcribe

# 小批量试跑：只分析某个账号的前 5 条
python douyin/import_videos.py douyin_videos --no-transcribe --account "狗狗执行官-Kiki" --limit 5
```

导入脚本会按 `video_id` 去重；如果同一条视频同时有 `.mp4` 和 `.mp3`，会优先选择已有转录的音频文件。点赞/互动数、互动等级、来源账号、频道分类、媒体路径和转录路径都会写入知识库 metadata。
