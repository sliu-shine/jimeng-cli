# 爆款文案智能体使用指南

## 系统概述

这是一个完整的抖音爆款文案智能体系统，能够：
1. 下载抖音爆款视频
2. 提取视频逐字稿
3. 分析爆款模式并存入知识库
4. 基于学习的模式生成新文案

## 快速开始

### 1. 安装依赖

```bash
# 基础依赖
pip install -r requirements_viral.txt

# 抖音下载器依赖
pip install -r requirements_douyin.txt
```

### 2. 运行完整流程测试

```bash
python test_full_pipeline.py
```

这个脚本会引导你完成：
- 下载抖音视频（可选）
- 提取逐字稿（Whisper 或 Groq）
- 分析爆款模式
- 导入知识库
- 测试文案生成

### 3. 启动 Web 界面

```bash
python viral_agent_ui.py
```

访问 http://127.0.0.1:7860 使用图形界面。

## 详细使用方法

### 方式一：命令行工具

#### 1. 下载抖音视频

```bash
# 下载用户的爆款视频（点赞 > 10万）
python douyin_cli.py download \
  --user-url "https://www.douyin.com/user/MS4wLjABAAAA..." \
  --output ./videos \
  --max-count 20 \
  --min-likes 100000
```

#### 2. 提取逐字稿

```bash
# 使用 Whisper 本地识别
python douyin_cli.py transcribe \
  --video-dir ./videos \
  --method whisper \
  --model base

# 使用 Groq API（更快）
python douyin_cli.py transcribe \
  --video-dir ./videos \
  --method groq \
  --groq-api-key "your-api-key"

# 使用云雾 API（推荐，速度快且稳定）
python douyin_cli.py transcribe \
  --video-dir ./videos \
  --method yunwu \
  --yunwu-api-key "your-yunwu-api-key"
```

#### 3. 完整流程（下载 + 转录）

```bash
# 使用 Whisper 本地识别
python douyin_cli.py pipeline \
  --user-url "https://www.douyin.com/user/MS4wLjABAAAA..." \
  --output ./videos \
  --max-count 10 \
  --min-likes 100000 \
  --method whisper \
  --model base

# 使用云雾 API（推荐）
python douyin_cli.py pipeline \
  --user-url "https://www.douyin.com/user/MS4wLjABAAAA..." \
  --output ./videos \
  --max-count 10 \
  --min-likes 100000 \
  --method yunwu \
  --yunwu-api-key "your-yunwu-api-key"
```

#### 4. 导入知识库

```bash
# 导入抖音样本到知识库
python import_douyin_samples.py ./videos
```

#### 5. 生成文案

```bash
# 命令行生成
python -m viral_agent generate "如何提升工作效率" --num 3

# 查看知识库统计
python -m viral_agent stats
```

### 方式二：Python API

```python
import asyncio
from douyin_downloader import DouyinDownloader, VideoTranscriber
from viral_agent import ViralAgent, KnowledgeBase

async def main():
    # 1. 下载视频
    async with DouyinDownloader(output_dir="./videos") as downloader:
        videos = await downloader.get_user_videos(
            sec_uid="MS4wLjABAAAA...",
            max_count=10,
            min_likes=100000
        )

        for video in videos:
            await downloader.download_video(video)

    # 2. 提取逐字稿
    transcriber = VideoTranscriber(method="whisper", whisper_model="base")
    transcript = await transcriber.transcribe("./videos/video.mp4")

    # 3. 分析并导入知识库
    from viral_agent.analyzer import ViralAnalyzer

    analyzer = ViralAnalyzer()
    analysis = await analyzer.analyze(transcript)

    kb = KnowledgeBase()
    kb.add_sample({
        "transcript": transcript,
        "metadata": {"source": "douyin"},
        **analysis
    })

    # 4. 生成文案
    agent = ViralAgent()
    scripts = await agent.generate(
        topic="如何提升工作效率",
        num_versions=3
    )

    for script in scripts:
        print(script["script"])

asyncio.run(main())
```

## 配置说明

### Whisper 模型选择

| 模型 | 大小 | 速度 | 准确度 | 推荐场景 |
|------|------|------|--------|----------|
| tiny | 39M | 最快 | 低 | 快速测试 |
| base | 74M | 快 | 中 | 日常使用 |
| small | 244M | 中 | 好 | 平衡选择 |
| medium | 769M | 慢 | 很好 | 高质量需求 |
| large-v3 | 1550M | 最慢 | 最好 | 专业场景 |

### Groq API

优势：
- 速度快（比本地 Whisper 快 10-20 倍）
- 免费额度充足
- 无需本地 GPU

获取 API key：https://console.groq.com/

### 云雾 API（推荐）

优势：
- OpenAI 兼容接口，使用 Whisper 模型
- 速度快且稳定
- 国内访问友好
- 支持中文优化

API 信息：
- Base URL: https://api.yunwu.ai/v1
- API Key: `your-yunwu-api-key`
- 模型: whisper-1

使用方法：
```bash
# 设置环境变量
export YUNWU_API_KEY="your-yunwu-api-key"

# 或在命令中直接指定
python douyin_cli.py transcribe \
  --video-dir ./videos \
  --method yunwu \
  --yunwu-api-key "your-yunwu-api-key"
```

### 抖音 API

使用的是开源项目 [DouYin_TikTok_Download_API](https://github.com/Evil0ctal/Douyin_TikTok_Download_API)

公共 API 地址：`https://api.douyin.wtf`

注意：
- 公共 API 有速率限制
- 建议自己部署获得更好的稳定性
- 需要真实的抖音用户链接

## 常见问题

### 1. Whisper 下载模型很慢？

首次使用会自动下载模型，可以手动下载：
```bash
# 下载 base 模型
python -c "import whisper; whisper.load_model('base')"
```

### 2. Groq API 报错？

检查：
- API key 是否正确
- 是否有网络连接
- 是否超出免费额度

### 3. 抖音 API 无法访问？

- 检查网络连接
- 尝试更换用户链接
- 考虑自己部署 API 服务

### 4. 知识库为空？

确保已导入样本：
```bash
# 查看统计
python -m viral_agent stats

# 手动导入测试样本
python test_viral_agent.py
```

### 5. 生成的文案质量不好？

- 增加知识库样本数量（建议 > 20 条）
- 确保样本质量高（真实爆款）
- 提供更具体的主题描述
- 调整生成参数（num_versions）

## 性能优化

### 1. 加速转录

- 使用 Groq API 代替本地 Whisper
- 使用较小的 Whisper 模型（base/small）
- 批量处理时使用异步并发

### 2. 减少内存占用

- 使用轻量级嵌入模型（all-MiniLM-L6-v2）
- 限制知识库大小（定期清理旧样本）
- 分批处理大量视频

### 3. 提升生成质量

- 增加高质量样本
- 使用更详细的主题描述
- 生成多个版本后人工筛选

## 进阶使用

### 自定义分析维度

编辑 `viral_agent/analyzer.py` 中的 prompt，添加自定义分析维度。

### 扩展知识库

```python
from viral_agent import KnowledgeBase

kb = KnowledgeBase()

# 添加自定义样本
kb.add_sample({
    "transcript": "文案内容...",
    "hook_formula": "痛点+解决方案",
    "conflict_type": "认知冲突",
    "metadata": {"source": "custom"}
})
```

### 批量导入

```python
import json
from pathlib import Path
from viral_agent import KnowledgeBase

kb = KnowledgeBase()

# 从 JSON 文件批量导入
samples_file = Path("samples.json")
with open(samples_file) as f:
    samples = json.load(f)

for sample in samples:
    kb.add_sample(sample)
```
