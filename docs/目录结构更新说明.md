## 🔧 技术细节

### 文件名清理
- **账号名**：最多 30 字符，移除非法字符
- **视频标题**：最多 50 字符，移除非法字符
- **非法字符**：`/ \ : * ? " < > |`

### 特殊情况
- 无法获取账号名 → 使用 `未知账号`
- 无法获取标题 → 使用 `视频_{video_id}`

### 代码位置
- 下载器：`douyin/douyin_selenium.py` 第 610-628 行
- 审核工具：`douyin/review_transcripts.py` 第 165-180 行
