# Sora 视频生成 - 快速开始

## 🚀 5 分钟快速上手

### 第一步：设置 API 密钥

```bash
export YUNWU_API_KEY="your-yunwu-api-key-here"
```

### 第二步：测试功能

```bash
python test_sora.py
```

你会看到：
```
✅ 所有测试通过！
   - 片段数量: 2
   - 总时长: 36 秒
```

### 第三步：启动 UI

```bash
python viral_agent_ui.py
```

浏览器访问：`http://127.0.0.1:7860`

### 第四步：生成视频

1. 进入 **"🎥 Sora 视频生成"** 标签页
2. 载入或粘贴文案
3. 点击 **"🎥 拆分为 Sora 2.0 视频提示词"**
4. 保存队列到文件（例如：`my_video.json`）
5. 执行队列：
   ```bash
   python sora_queue.py my_video.json
   ```

## 📝 示例文案

```
你知道猫咪为什么喜欢睡在你身边吗？

其实这不是偶然。当猫咪选择在你旁边睡觉时，
它是在用最直接的方式告诉你：我信任你。

因为睡眠是猫咪最脆弱的时刻，它愿意在你身边放松警惕，
说明它把你当成了最安全的港湾。

所以下次当你的猫咪蜷缩在你身边时，别打扰它，
这是它给你的最高礼遇。
```

## 🎬 生成结果

系统会自动：
1. 将文案拆分成 2 个片段（每个 18 秒）
2. 为每个片段生成专业的 Sora 提示词
3. 创建队列 JSON 文件
4. 提交到云雾 API
5. 下载生成的视频

## 📂 输出目录结构

```
outputs/
├── Sora视频片段01.mp4
└── Sora视频片段02.mp4
```

## 💡 提示

### Sora vs Seedance

- **Sora**: 真实感视频，适合纪实、生活场景
- **Seedance**: 动画风格，适合治愈、科普内容

### 最佳实践

1. **文案长度**: 每段 50-100 字为宜
2. **场景描述**: 越具体越好
3. **动作细节**: 描述清晰的动作和表情
4. **时长控制**: 单段建议 10-15 秒

### 常见问题

**Q: API 密钥在哪里获取？**
A: 访问云雾 AI 官网注册账号获取

**Q: 生成需要多久？**
A: 通常 5-10 分钟，取决于队列长度

**Q: 如何查看进度？**
A: 执行器会实时打印状态日志

**Q: 生成失败怎么办？**
A: 检查 API 密钥、网络连接和提示词内容
