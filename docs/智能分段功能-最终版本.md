## 🚀 使用方法

### 1. 启动 UI
```bash
python viral_agent_ui.py
```

### 2. 操作流程
1. **输入文案** - 手动输入或从已保存文案中载入
2. **智能分段** - 设置参数 → 点击"开始智能分段"
3. **生成提示词** - 选择频道 → 点击"生成视频提示词"
4. **导出** - 导出为新项目或导入到现有项目

### 3. 参数设置
- **目标时长**：10秒（推荐）
- **最大时长**：15秒（Seedance 2.0限制）
- **口播速度**：6字/秒
- **视觉风格**：频道A-温馨治愈系（默认）
- **视频比例**：16:9（默认）
- **模型版本**：seedance2.0fast（默认）
