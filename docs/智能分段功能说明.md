# 智能分段 + 提示词生成功能

## 概述

这是一个全新的独立功能模块，与现有的 Seedance/Sora 功能完全独立，采用两步式工作流：

1. **智能分段** - 按句子完整性分段文案
2. **提示词生成** - AI 理解上下文，生成连贯的视频提示词

## 核心特点

### 1. 智能分段 (text_segmenter.py)

- ✅ **考虑句子完整性** - 不会在句子中间切断
- ✅ **灵活时长控制** - 目标10秒，最长支持15秒（Seedance 2.0限制）
- ✅ **智能合并短句** - 避免过短的片段
- ✅ **自动拆分长句** - 超长句子在逗号处切分
- ✅ **实时验证** - 检测超长/过短片段并给出警告

**分段策略：**
```
1. 优先在句子边界（。！？）切分
2. 如果单句在10-15秒之间，保持完整
3. 如果单句超过15秒，在逗号处切分
4. 如果逗号片段仍超过15秒，强制按字数切分
```

### 2. 提示词智能体 (prompt_agent.py)

- ✅ **理解上下文** - 传入完整文案，理解叙事逻辑
- ✅ **生成连贯镜头** - 前后段落画面自然过渡
- ✅ **详细分镜元素** - 包含镜头、主体、动作、场景
- ✅ **时长匹配** - 提示词匹配每段的时长要求
- ✅ **可配置风格** - 支持自定义视觉风格

**提示词结构：**
```
段落1:
清晰日漫动画风格，16:9，电影级画面。[具体画面描述]

镜头：特写/中景/全景，推拉摇移
主体：画面核心元素
动作：具体动作描述
场景：环境、背景、氛围
```

## 使用流程

### 步骤1：输入文案

- 可以手动输入完整文案
- 或从"已保存的文案"中载入

### 步骤2：智能分段

**参数设置：**
- **目标时长**：每段的理想时长（默认10秒）
- **最大时长**：每段的最大时长（默认15秒，Seedance限制）
- **口播速度**：字/秒（默认6字/秒）

点击"开始智能分段"后，系统会：
1. 按句子完整性切分文案
2. 显示分段结果（时间轴、字数、预览）
3. 验证分段质量（检测超长/过短片段）

### 步骤3：生成提示词

**参数设置：**
- **视觉风格**：描述整体画面风格（默认：清晰日漫动画风格，16:9，电影级画面）
- **保持场景连贯**：是否让AI生成连贯的镜头过渡

点击"生成视频提示词"后，系统会：
1. 调用 Claude API
2. 理解完整文案的上下文
3. 为每个段落生成连贯的视频提示词
4. 显示提示词结果（包含分镜元素）

### 步骤4：导出

**两种导出方式：**

1. **导出为新项目** - 创建一个新的 Seedance 项目
2. **导入到现有项目** - 追加到已有项目的队列

**配置选项：**
- **项目名称**：留空则自动生成
- **视频比例**：9:16 / 16:9 / 1:1
- **模型版本**：seedance2.0 / seedance2.0fast / seedance2.0_vip / seedance2.0fast_vip

## 与现有功能的区别

| 特性 | 现有 Seedance 功能 | 新功能：智能分段+提示词 |
|------|-------------------|----------------------|
| 分段方式 | 按字数硬切分 | 按句子完整性智能分段 |
| 时长控制 | 固定时长 | 灵活时长（10-15秒） |
| 提示词生成 | 基于模板 | AI 理解上下文生成 |
| 场景连贯性 | 独立片段 | 连贯镜头过渡 |
| 独立性 | - | 完全独立，不影响现有功能 |
