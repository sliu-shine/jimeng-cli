# 抖音爆款视频采集使用指南

## 系统架构

```
浏览器 (Chrome + Tampermonkey)
    ↓
自动化脚本 (Selenium)
    ↓
视频下载 + 转录
    ↓
导入爆款智能体知识库
```

## 使用步骤

### 1. 安装 Tampermonkey 脚本

1. 在 Chrome 浏览器中安装 [Tampermonkey 扩展](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)

2. 打开 Tampermonkey 管理面板，点击「添加新脚本」

3. 复制 `douyin_downloader/tampermonkey_script.js` 的全部内容并粘贴

4. 保存脚本（Ctrl+S 或 Cmd+S）

### 2. 运行自动化采集

```bash
# 方式1: 使用测试脚本（推荐）
python3 test_douyin_selenium.py

# 方式2: 使用命令行工具
python3 douyin_cli.py
```

### 3. 操作流程

1. **脚本自动打开浏览器**
   - 访问指定的抖音用户主页
   - 自动滚动加载视频列表

2. **手动操作 Tampermonkey 面板**
   - 在页面右下角会出现一个浮动面板
   - 点击「🔍 扫描爆款视频」按钮
   - 脚本会自动扫描所有视频并按点赞数筛选
   - 确认视频列表无误

3. **确认下载**
   - 回到终端，按 Enter 键继续
   - 脚本会自动批量下载视频到 `douyin_videos/` 目录

4. **视频转录（可选）**
   ```bash
   # 使用 Whisper 本地转录
   python3 -c "from douyin_downloader.transcriber import transcribe_video; transcribe_video('douyin_videos/视频ID.mp4')"
   ```

5. **导入到爆款智能体**
   ```bash
   python3 import_douyin_samples.py
   ```

### 4. 配置说明

编辑 `test_douyin_selenium.py` 中的配置：

```python
config = {
    'user_url': 'https://www.douyin.com/user/...',  # 抖音用户主页链接
    'min_likes': 2000,   # 最低点赞数（在 Tampermonkey 中设置）
    'max_videos': 10,    # 最多下载数量
    'output_dir': './douyin_videos'
}
```

### 5. Tampermonkey 脚本功能

- **自动扫描**: 扫描当前页面所有视频
- **智能筛选**: 按点赞数过滤爆款视频（默认 > 2000）
- **数据存储**: 将视频列表存入 localStorage 供 Python 读取
- **可视化面板**: 实时显示扫描进度和结果

### 6. 常见问题

**Q: 浏览器打开后没有看到 Tampermonkey 面板？**
- 检查 Tampermonkey 扩展是否已启用
- 确认脚本已保存并启用
- 刷新页面重试

**Q: 点击「扫描视频」后没有反应？**
- 打开浏览器控制台（F12）查看错误信息
- 确认页面已完全加载
- 尝试手动滚动页面加载更多视频

**Q: 下载失败或视频无法播放？**
- 抖音视频链接可能有时效性
- 尝试使用其他下载方式（浏览器插件）
- 检查网络连接

**Q: 如何修改筛选条件？**
- 编辑 `douyin_downloader/tampermonkey_script.js`
- 修改 `MIN_LIKES` 变量（默认 2000）
- 保存后刷新页面

### 7. 完整工作流示例

```bash
# 1. 采集视频
python3 test_douyin_selenium.py
# → 浏览器打开 → 点击扫描 → 按 Enter → 自动下载

# 2. 查看下载结果
ls -lh douyin_videos/

# 3. 转录视频（如果需要）
# （Whisper 转录会在导入时自动进行）

# 4. 导入到智能体
python3 import_douyin_samples.py

# 5. 启动 Web 界面测试
python3 viral_agent_ui.py
# → 访问 http://127.0.0.1:7860
# → 在「生成文案」标签页测试
```

## 技术细节

### Selenium 配置
- 使用 webdriver-manager 自动管理 ChromeDriver
- 隐藏自动化特征（webdriver 检测）
- 自定义 User-Agent

### 数据流转
```
Tampermonkey 扫描
    ↓
localStorage 存储
    ↓
Selenium 读取
    ↓
批量下载到本地
    ↓
Whisper 转录
    ↓
ChromaDB 知识库
```

### 文件结构
```
douyin_videos/          # 下载的视频文件
├── 视频ID1.mp4
├── 视频ID2.mp4
└── metadata.json       # 视频元数据

.viral_kb/              # 爆款智能体知识库
└── chroma.sqlite3      # ChromaDB 数据库
```
