## 工作流程

### 自动模式（推荐）

1. **用户在 Web UI 添加账号**
   - 输入抖音用户主页链接
   - 配置下载参数（保存路径、最低点赞数、分类选项）

2. **点击"开始下载"**
   - Web UI 发送 POST 请求到 `/api/douyin/start`
   - 后端启动后台线程运行下载器

3. **下载器初始化**
   - 启动 Chrome 浏览器
   - 将账号队列注入到 localStorage

4. **自动处理每个账号**
   - 打开账号主页
   - 滚动加载视频
   - Tampermonkey 脚本自动扫描视频
   - 下载器轮询检查扫描状态
   - 下载视频并保存元数据
   - 自动跳转到下一个账号

5. **实时状态更新**
   - 下载器通过回调函数更新进度
   - Web UI 轮询 `/api/douyin/status` 获取最新状态
   - 显示进度条、日志和账号状态

6. **完成**
   - 所有账号处理完成
   - 关闭浏览器
   - Web UI 显示完成提示

## API 接口

| 方法 | 路径 | 功能 |
|------|------|------|
| GET | `/douyin` | 抖音采集页面 |
| GET | `/api/douyin/accounts` | 获取账号列表 |
| POST | `/api/douyin/accounts` | 保存账号列表 |
| GET | `/api/douyin/status` | 获取任务状态 |
| POST | `/api/douyin/start` | 启动下载任务 |
| POST | `/api/douyin/stop` | 停止下载任务 |

## 使用方法

### 快速开始

1. **安装依赖**
   ```bash
   pip install selenium webdriver-manager requests
   ```

2. **安装 Tampermonkey 脚本**
   - 在 Chrome 安装 Tampermonkey 扩展
   - 复制 `douyin/douyin_downloader/tampermonkey_script.js` 内容
   - 在 Tampermonkey 中创建新脚本并保存

3. **启动 Web 服务**
   ```bash
   python3 web_app.py
   ```

4. **访问抖音采集页面**
   ```
   http://127.0.0.1:8765/douyin
   ```

5. **添加账号并开始下载**
   - 点击"添加账号"输入链接
   - 配置下载参数
   - 点击"开始下载"
